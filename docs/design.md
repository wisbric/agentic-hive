# tmux html overlay

## I want

- use claude code anywhere via tmux, in a browser or via ssh directly (connecting to tmux)
- have one app that does it all as a webapp running in a container
- start new tmux sessions from the app
- being able to attach multiple tmux servers
- ssh keys / login needs to be encrypted (going via the public internet!)


## I have

- devbox vm provisioned on vdc-bootstrap in ionos (devbox.wisbric.com)
- this will run the claude code sessions via tmux
- has all required tools installed
